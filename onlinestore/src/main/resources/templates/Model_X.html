Descriere

Tesla Model X este cel mai nepoluant SUV din lume. Asteptarile clientilor si fanilor Tesla pentru versiunea SUV par a fi luat sfarsit. Fondatorul companiei Tesla, Elon Musk, a lansat acest model fara lasere si jocuri de lumini, fara animatori si spectacol, simplu si elocvent. Noul model vine pe piata cu cele mai performante sisteme de siguranta pasive si active, Automatic Emergency Braking si Side Collision Avoidance fiind oferite de la echiparea standard. Din punctul de vedere al caroseriei, Tesla Model X are cel mai bun coeficient de aerodinamicitate din istoria SUV-urilor: 0,24. Standard Model X vine cu 7 locuri, dar optional se poate livra si numai cu 6 locuri. Portierele din spate sunt tip „fluture” si sunt actionate electric. Tesla propune doua variante de motorizare: 90D cu 518 CP (distribuiti 259 CP puntea fata si 259 CP puntea spate prin cele doua motoare electrice) si P90D cu 762 CP (distribuiti 259 CP puntea fata si 503 CP puntea spate). Autonomia celor doua versiuni este de 413 km (P90D), respectiv 400 km (90D).
